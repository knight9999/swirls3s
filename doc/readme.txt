使い方

distディレクトリ以下に、ビルド済みのファイルinfo.nfuture.SRRE_0.6.0.0.jarがあります。
Eclipseのpluginsフォルダに、jarファイルを置いてください。
.rbファイルと.rhtmlファイルが対象となり、色づけされます。とくに、アシストなどの機能はありません。
パースペクティブは用意してありません。デフォルトでは、自動的にrailsプロジェクトには赤いマークがつ
きます。(/config/environment.rbファイルがあると、railsプロジェクトとみなします)　プロジェクトの
プロパティから、自動でrailsプロジェクトかどうかを判断するかどうかの選択ができます。
